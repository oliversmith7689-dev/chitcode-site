# ЧИТКОД — Алгоритмическое продвижение в Telegram

Landing Page для сервиса «PulseGrowth» (ЧИТКОД), специализирующегося на органическом продвижении Telegram-каналов с использованием уникальной технологии «Лесенки».

## Особенности
- **Технологичный дизайн**: Использование абстрактной графики, глитч-эффектов и элементов «цифрового дождя».
- **Интерактивные графики**: Визуализация распределения активности (72 часа) с помощью D3.js.
- **Адаптивность**: Полная поддержка мобильных устройств и десктопов.
- **Безопасность**: Акцент на анонимности и отсутствии меток в сервисах аналитики (TGStat/Telemetr).

## Стек технологий
- React 19
- Tailwind CSS
- D3.js (для графиков и визуализаций)
- Google Gemini API (для интеллектуальных функций, если применимо)

## Развертывание
Проект представляет собой SPA (Single Page Application) на базе ES-модулей. Основные файлы: `index.html`, `index.tsx` и `App.tsx`.
